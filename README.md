## 📌 Descripción

Este es mi portafolio personal como desarrollador de software, diseñado para mostrar mis habilidades, proyectos y experiencia en el desarrollo de sistemas modernos y escalables.

El proyecto está enfocado en un diseño tipo landing page profesional con estilo minimalista y moderno, inspirado en interfaces tipo Apple.

---

## ✨ Características

- 🎨 Diseño moderno tipo landing page
- 🌙 Modo claro / oscuro
- 🧑 Avatar personal en el hero
- 📄 Botón de descarga de CV
- 🖱️ Cursor personalizado
- 🌌 Fondo animado tipo gradiente
- 📱 100% responsive (mobile-first)
- ⚡ Animaciones suaves y UX optimizada

---

## 🧠 Secciones del portafolio

- **Hero** → Presentación principal + CV
- **About** → Información personal y enfoque profesional
- **Skills** → Tecnologías y herramientas
- **Projects** → Proyectos destacados
- **Footer** → Enlaces y contacto

---

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3 (Flexbox + Animaciones)
- JavaScript Vanilla
- Three.js (fondo 3D)
- Google Fonts (Poppins)
